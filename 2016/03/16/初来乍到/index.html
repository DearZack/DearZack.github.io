初来乍到

2016-03-16 • 标签: 初来乍到, 博客搭建

发篇文章纪念一下，昨天搭建好以后，挺激动的，我终于有自己的博客了！

激动之余，也感叹现在大伙的开源精神，一个完全没有H5知识的我也可以拥有这么优美的网页。

顺便将一下在Mac下怎么搭建。

1.下载node.js

node.js 官网地址 https://nodejs.org/en/
node -v
npm -v
如果有输出版本信息则说明安装成功

2.下载hexo

sudo npm install hexo –no-optional
最好加上sudo，不然后可能没有权限在磁盘写文件
hexo -v
如果有输出信息就说明安装成功了

3.常用指令

hexo init
hexo clean
hexo g
hexo s
hexo d
